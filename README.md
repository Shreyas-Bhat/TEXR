## Pseudolabeling

### Vanilla Pseudolabeling with TabSTAR

Trains TabSTAR to predict each feature from all other features, generating pseudolabels and confidence scores.

```bash
python pseudolabeling/vanilla_pseudolabel_tabstar.py \
    --dataset_dir /path/to/datasets \
    --output_dir  /path/to/output \
    --pretrain_path /path/to/tabstar/checkpoint \
    --device cuda:0 \
    --mode lora \          # 'lora' or 'fast'
    --n_shots 0 \
    --max_epochs 10 \
    --patience 3 \
    --dataset_pattern "*.csv" \
    --max_datasets 0       # 0 = all datasets
```

Output: one `{dataset}_pseudolabels.csv` per dataset with `{feature}_pseudo` and `{feature}_confidence` columns.

---

### CAST Pseudolabeling with TabSTAR

Autoregressive CAST pseudolabeling using TabSTAR. Supports both `vanilla` and `cast` methods via `--method`.

```bash
python pseudolabeling/cast_pseudolabel_tabstar.py \
    --data_dirs /path/to/data1 /path/to/data2 \
    --output_dir /path/to/output \
    --method cast \                  # 'vanilla' or 'cast'
    --pretrain_path /path/to/tabstar/checkpoint \
    --device cuda:0 \
    --n_shots 5 \
    --max_epochs 10 \
    --patience 3 \
    --cast_rounds 3 \
    --confidence_threshold 0.85 \
    --n_clusters 8 \
    --cast_k 10 \
    --cast_alpha 1.0 \
    --max_datasets 0       # 0 = all datasets
```

Alternatively, pass a file listing data directories:

```bash
python pseudolabeling/cast_pseudolabel_tabstar.py \
    --data_dirs_file /path/to/dirs.txt \
    --output_dir /path/to/output \
    --method cast
```

### General Pseudolabeling (`pl.py`, ASPIRE-based)

Pseudolabeling using an ASPIRE-style model. Supports both vanilla and CAST retrieval.

```bash
python pseudolabeling/pl.py \
    --checkpoint /path/to/aspire/checkpoint \
    --output_dir /path/to/output \
    --data_dirs /path/to/data1 /path/to/data2 \
    --method cast \                  # 'vanilla' or 'cast'
    --device cuda:0 \
    --temperature 3.0 \
    --cast_k 10 \
    --cast_alpha 1.0 \
    --cast_metric cosine \           # 'cosine' or 'euclidean'
    --cast_index_rows 30 \
    --cast_max_per_class 30 \
    --model_module aspire_enhaced_v2_icml \
    --model_class ASPIREEnhanced \
    --max_datasets 0
```

## Training

### Pretrain TabSTAR on CAST Synthetic Data

Trains a TabSTAR foundation model from scratch using CAST-generated pseudolabeled datasets as the pretraining corpus.

```bash
python training/pretrain_tabstar_cast.py \
    --dataset_dir /path/to/cast/pseudolabels \
    --output_dir /path/to/checkpoints \
    --device cuda:0 \
    --batch_size 32 \
    --global_batch_size 256 \
    --max_epochs 10 \
    --learning_rate 1e-4 \
    --weight_decay 0.01 \
    --patience 3 \
    --tabular_layers 6 \
    --unfreeze_layers 3 \
    --val_ratio 0.2 \
    --dataset_pattern "*.csv" \
    --max_datasets 0
```

### Train CM2 on CAST Pseudolabeled Data

Fine-tunes CM2 on pseudolabeled CSVs generated by CAST.

```bash
python training/train_cm2_vllm_pseudolabels_cast.py \
    --data_dir /path/to/pseudolabels \
    --save_dir ./checkpoints/cm2_cast \
    --pretrained_checkpoint /path/to/cm2/pretrained \
    --num_epochs 40 \
    --batch_size 32 \
    --lr 1e-4 \
    --confidence_threshold 0.0 \
    --max_datasets 5000 \
    --max_examples_per_dataset 100 \
    --device cuda:0
```

### Evaluate TP-BERTa Multi-Seed

Runs evaluation across multiple random seeds and reports mean ± std.

```bash
python evaluation/evaluate_tpberta_cast_multiseed.py \
    --pretrain_dir /path/to/tpberta/pretrained \
    --model_suffix pytorch_models/best \
    --settings few_shot finetuning \
    --num_shots 5 \
    --fewshot_epochs 50 \
    --finetune_epochs 100 \
    --lr 1e-4 \
    --batch_size 32 \
    --output_dir ./tpberta_cast_multiseed_results \
    --seeds 42 123 456 \
    --gpu 0 \
    --parallel \           # run seeds in parallel
    --gpus 0 1 2           # one GPU per seed (only with --parallel)
```
